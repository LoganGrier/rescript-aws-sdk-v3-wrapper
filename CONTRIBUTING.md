## Requirements

* git
* yarn
* nodejs >= 12

## Setup

1. Clone the repository i.e. 

    ```sh
    git clone git@github.com:chris-armstrong/rescript-aws-sdk-v3-wrapper.git
    ```
2. Initialise the submodules

    ```sh
    git submodule init
    git submodule update
    ```

3. Build generator and then generate definitions

    ```sh
    yarn regenerate
    ```
